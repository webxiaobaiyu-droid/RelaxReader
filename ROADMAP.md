# Relax Reader · 产品路线图

> 目标：从"能装一本书的阅读器"演进到"能持续读到想读的书的阅读平台"。
> 闭环优先，体验其次，花哨功能最后。

---

## L0 · 已完成（v1.0）

- 本地 TXT 导入 + 自动编码识别（UTF-8 / UTF-16 / GBK）
- 中英文章节自动切分
- 三种阅读模式（滚动 / 滑动 / 翻页）
- 阅读设置（主题、字体、字号、行高、页宽、页边距）
- 目录、进度、自动滚动、快捷键
- 在线书源（单源 · xbiqugu）+ 通用 URL 导入

---

## L1 · 书源闭环（本期已落地）

让"找书 → 抓取 → 阅读"的链路不再单点失效。

| 模块 | 状态 | 说明 |
|---|---|---|
| 书源数据模型 + 持久化 | ✅ | 内置 + 自定义统一存储，含 enabled / priority / health |
| 书源管理 UI（齿轮入口） | ✅ | 启用开关、优先级、测试、添加、编辑、删除 |
| 多源并发搜索 | ✅ | `Promise.allSettled`、按 title+author 去重 |
| 健康度自动统计 | ✅ | 实时记录成功/失败、平均延迟、错误信息 |
| 失败可见 | ✅ | 搜索结果显示各源耗时；齿轮按钮异常红点 |
| 书源 JSON 导入/导出 | ✅ | 单文件分享 |
| 错误隔离 | ✅ | 单源失败不影响其他源；具体错误回传 UI |

## L2 · 内容闭环（下一期，3–4 周）

让"加入书架的书 = 永远能读到 + 永远是最新的"。

### L2.1 本地章节缓存（最高优先级）

**问题**：当前在线书每次重开都重抓，弱网/离线直接失能。

**方案**：
- 主进程 SQLite（`better-sqlite3`）或 `electron-store` 分文件存
- 表结构：`chapter_cache(book_id, chapter_index, content_text, fetched_at, source_id)`
- 抓取后立刻落盘；ReaderView 优先读缓存
- 设置页加「缓存大小 / 清理缓存」入口
- 预加载策略：当前章 ±2 章后台抓取

**验收**：飞机模式下能继续读已加载过的章节。

### L2.2 追更（订阅式更新提醒）

**问题**：在线书加入书架后，新章节用户感知不到。

**方案**：
- 主进程每 N 分钟（可配置 30/60/120/手动）轮询启用源的目录页
- 比对本地 `totalChapters`，新章节数 > 0 时持久化 `newChaptersCount`
- 书架卡片角标 + 桌面通知（可选）
- 用户点开书时自动滚动到第一个未读章节

**验收**：xbiqugu 上有新章节时，1 小时内书架显示红点。

### L2.3 换源

**问题**：源 A 章节缺失/被删时，没法切到源 B 接着读。

**方案**：
- 同一本书可关联多个源（多 `bookUrl`）
- 章节级模糊匹配（标题相似度 ≥ 0.7 视为同一章）
- 阅读器顶部加「切换书源」菜单
- 进度按"章节索引 + 章内偏移"映射（章长不一致时取百分比）

**验收**：在源 A 读到 100 章，切到源 B 能从 101 章开始。

### L2.4 阅读进度云同步（暂不做）

> 需要后端，本期范围外。先把本地体验做扎实。

## L3 · 体验加强（下下期，2–3 周）

### L3.1 订阅式书源市场

- 远程 JSON URL（社区维护，如 `https://example.com/sources.json`）
- 含 `version`、`updatedAt`、`sources[]`
- 应用启动后台静默更新；新增源以"推荐"状态默认禁用
- 用户可在书源管理页订阅多个 URL

### L3.2 正文净化规则（每源可配）

- 通用净化：广告关键词黑名单（"本章未完"、"加入书签"、域名残留）
- 正则替换规则：每源自定义 `{ pattern, replace }[]`
- 段落智能合并：识别误切的句子

### L3.3 阅读数据

- 阅读时长（按书、按天）
- 翻页/章节统计
- 阅读速度（字/分钟）→ 用于 ReaderView 的 `≈时间估算` 更准
- 数据本地存储，不上传

### L3.4 书架功能升级

- 分组（默认/收藏/已完结/待读）
- 标签
- 排序（最近阅读 / 添加时间 / 标题 / 完结状态）
- 搜索本地书架

---

## L4 · 长期方向（不承诺时间）

- EPUB 支持（已在 README 提到，但未实现）
- 多窗口（同时读两本）
- 听书（系统 TTS）
- 笔记 / 划线 / 书签（含搜索）
- 自定义快捷键
- 移动端（iOS/Android · Capacitor 或重写）

---

## 非目标（明确不做）

- ❌ 用户账号系统（隐私优先，纯本地）
- ❌ 内嵌 IM/社区/书评（偏离阅读核心）
- ❌ 内购 / 会员体系（开源 + 免费）
- ❌ 主动推送广告（破坏阅读沉浸）

## 优先级速查表

| 优先级 | 内容 | 解决什么问题 |
|---|---|---|
| P0 | L2.1 本地缓存 | 离线读不了 |
| P0 | L2.2 追更 | 加书架后无更新感知 |
| P1 | L2.3 换源 | 源失效后无法接续 |
| P1 | L3.1 订阅书源 | 新源传播慢 |
| P2 | L3.2 净化规则 | 抓取内容有杂质 |
| P2 | L3.4 书架分组 | 书多了找不到 |
| P3 | L3.3 阅读数据 | 没有反馈感 |
| P3 | L4 EPUB 支持 | 格式单一 |
